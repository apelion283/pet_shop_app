## üìåAbout The Project

Pet Shop is a user-friendly e-commerce app for pet lovers, offering seamless browsing, purchasing, and management of pet products, accessories, and food. It features secure authentication, real-time notifications, wishlist, store locator, and a smooth checkout process.

Built with Flutter, Bloc-Cubit, Hive, and Firebase, the app ensures high performance, security, and scalability with a Clean Architecture approach. It supports multi-language detection, optimizing the shopping experience for a global audience.

### üéØFeatures  
üîê **Authentication & Security**: User authentication with login, logout, signup, and password recovery.  
üêæ **Pet Marketplace**: Browse, add to cart, purchase, and share pet products, including pets, pet food, and accessories via app links.  
üó∫Ô∏è **Store Locator**: Display the user‚Äôs current location on Google Maps and nearby pet stores.  
üõí **Seamless Checkout**: Add custom messages to orders when placing an order.  
üë§ **Account Management**: Update profile details, including name, password, and avatar.  
üîî **Notification Preferences**: Configure and manage to receive notifications from server-based push notifications.  
‚ú® **Optimized UX**: Enhanced user experience with shimmer effects for loading states, interactive loading indicators, and real-time response feedback.  
üìå **Scrollable List Synchronization**: Explore page tabs stay synchronized while scrolling for a seamless navigation experience.  
üåç **Multi-Language Support**: Automatically detect the device language. If it is not **English** or **Vietnamese**, the app defaults to **Vietnamese**. Otherwise, it follows the system language settings, ensuring a seamless and localized user experience.  
‚ù§Ô∏è **Favorite List**: Users can add products to their wishlist for easy access later. The wishlist screen allows users to **add**, **remove**, and **move products to the cart** seamlessly, providing a convenient shopping experience.
